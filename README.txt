                        MCQuick
			=======

What is MCQuick?
----------------

MCQuick is a web application for creating and solving MCQ papers.

MCQuick is written in PHP and Javascript for server installation.

Downloading
-----------

Latest release of MCQuick can be downloaded from http://code.agnibho.com/mcquick/

The source code is available at https://github.com/agnibho/mcquick/

Installation
------------

Please see the file named INSTALL.txt for installation instructions.

Usage
-----

After installation MCQuick can be accessed from the designated
URL. Specific usage instructions are available in the help.php page of
the MCQuick.

A hosted version of MCQuick is available at
http://app.agnibho.com/mcquick/
