=============
MCQuick 1.0.1
=============

General Usage Notes
-------------------
-MCQuick can be used to solve MCQs and check answers
-MCQuick produces blank answer sheets which can be used to solve MCQs from a book or seperate question paper.
-MCQuick can also open specially prepared question papers for solving
-No download is required.
-Javascript must be available to use MCQuick.

Program Notes
-------------
-MCQuick is programmed in Java Script
-MCQuick is coded in an Object Oriented fashion
-Bootstrap and jQuery are used, which are included from their CDN
-tv4 json validation library is used (https://github.com/geraintluff/tv4)
-The source code of MCQuick is available at https://github.com/agnibho/mcquick
